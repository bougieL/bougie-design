## Introduction
A UI Design Language based on TypeScript.

## Installation
#### use yarn
```bash
yarn add bougie-design
```
or
#### use npm
```bash
npm i bougie-design -D
```

## API Documentation
### scripts

#### `yarn start`
Run the app at development mode.

#### `yarn build`
Build the project.

#### `yarn lint`
Code style lint

#### `yarn fix`
Code style fix
