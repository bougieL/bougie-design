# Icon

Semantic vector graphics.

## When To Use

A button means an operation (or a series of operations). Clicking a button will trigger corresponding business logic.

## Examples

```jsx
import { Icon } from 'bougie-design'
```

## Api

| Property | Description                                                   | Type                                                 | Default   |
| :------- | :------------------------------------------------------------ | :--------------------------------------------------- | :-------- |
| disabled | disabled state of button                                      | boolean                                              | `false`   |
| icon     | set the icon of button                                        | string                                               | -         |
| type     | can be set to `default` `primary` `warning` `error` `success` | string                                               | `default` |
| onClick  | set the handler to handle `click` event                       | `(evt: React.MouseEvent<HTMLButtonElement>) => void` | -         |
